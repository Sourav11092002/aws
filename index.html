Q1: What is AWS?

AWS stands for Amazon Web Services. It is a cloud computing platform provided by Amazon that offers services such as computing power, storage options, and networking capabilities.

Q2: What is EC2?

EC2 stands for Elastic Compute Cloud. It allows users to rent virtual machines to run their applications.

Q3: What is S3?

S3 stands for Simple Storage Service. It is used to store and retrieve any amount of data at any time from anywhere on the web.

Q4: What is IAM in AWS?

IAM stands for Identity and Access Management. It helps you securely control access to AWS services and resources for your users.

Q5: What is AWS Lambda?

AWS Lambda is a serverless compute service that lets you run code without provisioning or managing servers.

Q6: What is the use of Route 53?

Amazon Route 53 is a scalable and highly available Domain Name System (DNS) web service used to route users to internet applications.

Q7: What is an AWS VPC?

VPC stands for Virtual Private Cloud. It allows you to launch AWS resources into a virtual network that you've defined.

Q8: What is CloudFormation?

AWS CloudFormation is a service that helps you define and provision AWS infrastructure using a declarative template.

Q9: What is Auto Scaling?

Auto Scaling helps you automatically adjust the number of EC2 instances in your application based on demand.

Q10: What is Amazon RDS?

Amazon RDS (Relational Database Service) is a managed database service that supports various database engines such as MySQL, PostgreSQL, Oracle, and SQL Server.
